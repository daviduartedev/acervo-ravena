# acervo-ravena

Site institucional e acervo documental do **Ravena Cassino Hotel** (Laguna - Santa Catarina).

## Stack

- Next.js 16 (App Router, SSG)
- React 19
- Tailwind CSS 4
- TypeScript

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```

## Estrutura

- `app/` — rotas e layout
- `components/` — seções e UI
- `content/` — textos, cronologia, acervo e publicações documentadas
- `public/acervo/` — fotografias do acervo
